# 🏡 Site de Locação de Espaço para Eventos

## 📌 Sobre o Projeto

Este projeto consiste em um site desenvolvido para divulgação de um espaço para locação de eventos.
O objetivo é apresentar o ambiente de forma atrativa, facilitando o contato com possíveis clientes e destacando os principais diferenciais do local.

---

## 🚀 Funcionalidades

* Página inicial moderna e responsiva
* Página de detalhes do espaço
* Página “Saiba Mais” com descrição completa
* Galeria de imagens interativa
* Botão de contato direto via WhatsApp
* Animações e efeitos visuais modernos
* Layout inspirado em sites profissionais (ex: Airbnb)

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
